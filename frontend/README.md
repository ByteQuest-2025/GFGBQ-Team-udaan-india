
  # Healthcare Dashboard Design

  This is a code bundle for Healthcare Dashboard Design. The original project is available at https://www.figma.com/design/s2ZAHYWChGcSpNPywpB6fS/Healthcare-Dashboard-Design.

    ## Running the code

    1. Install dependencies:

      `npm install`

    2. Configure the API base URL via environment variables:

      - Create a `.env.development` file with:

       `VITE_API_BASE_URL=http://127.0.0.1:8000`

      - For production builds, set `VITE_API_BASE_URL` in `.env.production` or the deploy environment.

    3. Start the development server:

      `npm run dev`

    4. Build for production:

      `npm run build`
